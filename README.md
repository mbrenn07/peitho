# Setup Instructions

## Chrome Extension

Ensure you have React's most recent LTS version installed (may work with other versions)

cd `chrome-extension`

run `npm install`

run `npm run build`

Then, open `chrome://extensions/` and enable `developer mode`

Finally, `load unpacked` and select the `build` folder

## Python Server

pip install:
- flask
- flask_cors
- bs4

then run start.sh
